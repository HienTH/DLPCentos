3. Khởi tạo MegaDLPService: cp megaDLPService.service /etc/systemd/system/megaDLPService.service

Thêm nội dung vào file service:

[Unit]
Description=Server Information Service
After=network.target

[Service]
ExecStart=/opt/MegaDLP/megaDLPService
Restart=always

[Install]
WantedBy=multi-user.target

Execute command:
sudo systemctl daemon-reload
sudo systemctl enable megaDLPService
sudo systemctl start megaDLPService
